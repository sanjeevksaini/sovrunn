# Reuse Philosophy

Document
- ID: reuse-philosophy
- Version: 1.0
- Status: Stable

Purpose
- Define adoption philosophy

Beliefs

Knowledge
- Adopt before discover

Specification
- Adopt before define

Architecture
- Adopt before design

Algorithm
- Adopt before invent

Implementation
- Adopt before build

Ownership
- Own only differentiated capability

Differentiation

Sovrunn SHOULD own
- Semantic model
- Semantic planning
- Semantic execution
- Cross engine interoperability
- Unified capabilities
- Autonomous intelligence

Sovrunn SHOULD NOT own
- Commodity standards
- Commodity protocols
- Commodity algorithms
- Commodity infrastructure
- Commodity implementations

Success
- Maximum interoperability
- Minimum ownership
- Maximum reuse
- Sustainable evolution
